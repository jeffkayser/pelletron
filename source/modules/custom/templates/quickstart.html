Quickstart

Change front-end framework

  • Use Bootstrap
    • python site.py use -f bootstrap
  • Use Foundation
    • python site.py use -f foundation
  • Use HTML5 Boilerplate
    • python site.py use -f html5boilerplate
  • Use Pelletron default (no framework)
    • python site.py use -f custom

Build static site

  • python site.py build
Note: Previously built files are deleted during the build process. There's no need to remove them manually.

Deploy static site

  • python site.py deploy -h host -u username -d directory
